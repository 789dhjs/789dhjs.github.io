使用了 MATLAB 的冒号运算符 :，它生成一个从 0 到 2*pi 的数值序列。这里，pi/10 是步长，表示从 0 到 2*pi 之间的数值间隔为 pi/10。

例如，如果 x 向量是 [0, pi/10, 2*pi/10, ..., 2*pi]，那么 y 就是对应的正弦值：[sin(0), sin(pi/10), sin(2*pi/10), ..., sin(2*pi)]。

注意：在MATLAB中π用pi表示

MATLAB命令行既要一行一条命令，又可一行输入若干命令，各命令之间以逗号分隔，或者用 ； 标点符号分隔，两者的区别是 “,”相隔的返回执行结果，”;”相隔的不会返回执行结果 命令行后
% 标点符号，可用于注释或说明命令的含义，对命令执行结果不产生任何影响。
