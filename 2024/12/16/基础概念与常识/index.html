基础概念与常识

Java语言有哪些特征

1）oop(封装、继承、多态)

2）跨平台

3）多线程

4）可靠

5）安全

6）易上手

7）高效

JavaSEVS JavaEE

​ JavaSE 是JavaEE的基础版 JavaSE适合开发桌面应用程序或者是简单的服务器应用程序,而JavaEE支持了各种企业级开发程序和部署的标准和规范(如Servlet,Jsp,JDBC等)。JavaEE适用于构建Web应用程序

JVMVS JDKVS JRE

​ JVM 是可以运行字节码的一个虚拟机，也是Java实现"一次编译，随处可以运行"的关键所在。(将不同平台上产生的.class文件转化为相同的二进制文件)

JDK 是为Java开发工程师提供的一个开发工具包，用于创建和编译Java程序。

包含：1.JRE

​ 2.javac 等

JRE 只包含运行Java程序所需要的库和环境

包含：1.JVM

​ 2.Java基础类库(Class Library ):一组标准的类库，提供一些常用的库和API(如 I/O、网络编程、数据结构 等)

注意 :

JDK > JRE > JVM

Java编译过程

​ 先由javac编译器将.java文件编译成.class文件，再有JVM 虚拟机将.class文件进行解释在解释的过程中由JIT编译为二进制文件。(所以Java是半编译半解释语言)

Java 与 C++ 的区别

相同点：

面向对象，都用拥有面向对象的三大基本特征(封装，继承，多态)

不同点:

Java : 单继承，没有指针，不能进行引用传递(因为没有指针)，拥有垃圾回收机制

C++:多继承，支持指针，支持方法重载和和操作符重载。

标识符

关键字

​ Java官方将一些标识符赋予特殊的含义，在特定的场景中使用，是特殊的标识符
